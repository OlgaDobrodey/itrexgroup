1. labyrinth's schema is located in INPUT.TXT (src/com/company/Task2/INPUT.TXT)
2. The first line of INPUT.TXT file contains digits H, M, N - count of levels and rectangle size of each level.
Then other lines contain H blocks, each of them describes one level from top to bottom.
Each block contains M rows and N columns. Every cell contains: “.”, “o”, “1”, 2”.

   “.” means that it’s free space.    -char
   “o” means columns there.           -char
   “1” Prince position.               -char
   “2” Princes position.              -char
3.   Symbol “1” is always in the first block. Symbol “2” is always in the last block.
 Nearby blocks are separated by one blank line.
